Agrega al archivo html las Siguientes dependencias:

CSS - Bootstrap (v-3.4.1), CIDI, Iconos

<link src="https://lalala.com/CIDI.css" rel="stylesheet">

JavaScript - Bootstrap (v-3.4.1)

<script src="https://stackpath.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>
